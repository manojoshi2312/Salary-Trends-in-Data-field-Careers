# Salary Trends in Data Field Careers

## Description
This project analyzes salary trends across data-focused roles such as Data Analyst, Data Scientist,
and Data Engineer. The analysis focuses on understanding how experience level, job role, location,
and technical skills affect salary distribution.

## Objectives
- Identify high-paying roles in the data domain
- Analyze salary variation by experience and location
- Explore the relationship between skills demand and compensation

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Excel
- Power BI (for visualization)

## Key Insights
- Senior-level data roles show significantly higher salary ranges
- Certain technical skills strongly correlate with higher pay
- Location plays a major role in salary variation
